# Error Codes

## Code

| code | message                            |
| ---- | ---------------------------------- |
| 200  | Success                            |
| 400  | Param error                        |
| 401  | Authentication failed              |
| 404  | Path Not Found                     |
| 500  | Server error                       |
| 4001 | Date format is incorrect           |
| 4002 | Date can not be greater than today |
| 4003 | Only support nearly 30 reward data |
